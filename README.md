# Date Plann App

デートやお出かけのプラン作成を自動化するアプリです。  
ユーザーの条件（場所・時間・好み）をもとに、最適なルートとスポットを提案します。

---

## 概要

「どこに行くか決められない」「効率よく回りたい」といった課題を解決するために開発しました。  

本アプリでは、複数の候補スポットから移動コスト（距離・時間）を考慮し、最適な順序で訪れるデートプランを自動生成します。

---

## ターゲット

- デートプランを考えるのが苦手な人  
- 観光やお出かけルートを効率化したい人  
- 限られた時間で最大限楽しみたい人  

---

## 主な機能

## 1. デートプラン自動生成
- 出発地・目的地・滞在時間を入力  
- 条件に応じたスポットを抽出  
- 最適な順序でルートを生成  

---

## 2. ルート最適化
- スポットをノード、移動時間をエッジとしたグラフ構造で管理  
- ダイクストラ法（最短経路アルゴリズム）を用いて最適ルートを算出  
- 無駄な移動を削減し、効率的なプランを生成  

---

## 3. カスタマイズ機能
- プランタイプ選択（例：ゆったり / アクティブ）  
- 屋内・屋外の指定  
- 予算設定  

---

## 4. スポット提案
- カフェ・レストラン・観光地などをカテゴリ別に表示  
- 時間帯に応じたおすすめスポットを提示
